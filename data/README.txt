This folder contains the following items:
- Ini_dist_among_pops.txt = geographic distances among Ae. triuncialis populations
- Ini_dist_to_fields.txt = geographic distances among Ae. triuncialis populations and cultivations
- Ini_real_phen_freqs.txt = empirical frequencies of AFLP bands, computed with scripts in ../AFLP/
- Ini_sampling_scheme.txt = empirical sampling scheme
- Ini_template_file.txt = template INI file for QuantiNemo
